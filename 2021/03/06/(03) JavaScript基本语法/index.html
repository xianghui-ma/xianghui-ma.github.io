JS引入方式

JS有如下两种引入方式：

  • 页面级JS：页面内嵌<script></script>标签
  • 外部引入：<script src="location"></script>

JS基本语法

大小写

JS区分大小写

变量

  • 变量命名规则：变量名由数字，字母，下划线，$组成。但只能以字母，下划线，$开头
  • var a叫变量声明；var a = 10叫变量声明并赋值
  • 可以用单一var模式定义变量。如var a=10, b=20, c;
  • 类似于var a = b = c = 10这种连等形式，其中变量b和c均未声明，仅a被声明

值类型

JS中有两大类值类型：

  • 原始值：Number, String, Boolean, undefined, null
  • 引用值：包括Array, Function......
  1. 原始值存于stack中；而引用值的存储方式是：先在heap中开辟存储空间存储对象，然后将在heap中开辟的空间的地址放于stack中，从而指向heap中的区域。而无论是原始值还是引用值变量间的赋值，其都是将一个变量在stack中的值copy一份给予另一个变量
  2. null与undefined都是各自类型中唯一一个值。变量声明后未赋值则值为undefined
  3. 由于null最开始用于给对象占位，因此typeof(null)返回object

错误

JS中的错误（包括语法与逻辑错误）会使同一代码块中的后续代码中止执行，但不影响其他JS代码块

上述“代码块”是指<script></script>

运算符

这里仅探究一些运算符！

  • 比较运算符

  • 比较运算符包括>, >=, <, <=, ==, !=, ===, !==

  • 比较运算符的结果为boolean值

  • 除NaN!=NaN及NaN!==NaN外，其他涉及NaN的比较操作结果都为false

  • 逻辑运算符

  • 逻辑运算符包括&&, ||, !

  • 假值有：undefined, null, 0, "", false, NaN

  • &&属于短路操作。如果第一个操作数为假值则返回第一个操作数的值，无视第二个操作数；如果第一个操作数为真值则返回第二个操作数的值

  • ||属于短路操作。如果第一个操作数为真值则返回第一个操作数的值，无视第二个操作数；如果第一个操作数为假值则返回第二个操作数的值

  • !返回的是boolean值

  • 三目运算符

三目运算符的形式为表达式 ? 表达式1 ： 表达式2。如果表达式为真值则返回表达式1的值，否则返回表达式2的值

typeof操作符

  • typeof的返回值为：number, string, boolean, undefined, object, function
  • typeof的运算结果为string类型，故typeof(typeof(...))的结果为string
  • typeof(a),如果程序中没有a变量该运算也不会报错，而是返回undefined

NaN(Not a Number)

该值用来表示本来应返回数值但却未返回数值的情况。如：0除以0会返回NaN

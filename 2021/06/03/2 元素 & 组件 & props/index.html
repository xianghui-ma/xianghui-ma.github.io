组件

1、为什么需要组件

组件允许你将 UI 拆分为独立可复用的代码片段

2、组件的定义

React提供了两种方式来定义组件：一是，函数组件；二是，类组件

函数组件本质就是JS函数，但不同的是其一般接收 “props”（代表属性）参数且并返回一个 React 元素

类组件必须继承React.Component

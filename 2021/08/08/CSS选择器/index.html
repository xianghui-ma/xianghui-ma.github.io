CSS选择器

发表于 2021-08-08 更新于 2021-07-16 分类于 CSS
本文字数： 1.2k 阅读时长 ≈ 1 分钟

基础选择器

  • tagname——类型选择器或者标签选择器。该选择器匹配目标元素的标签名。例如：p、h1、strong
  • .class——类选择器。该选择器匹配 class 属性中有指定类名的元素。例如：.media、.nav-menu
  • #id——ID 选择器。该选择器匹配拥有指定 ID 属性的元素。例如：#sidebar
  • *——通用选择器。该选择器匹配所有元素

组合器

组合器将多个基础选择器连接起来组成一个复杂选择器，如.parent > .child

  • 子组合器（A > B）——匹配的目标元素B是A的直接后代。例如：.parent > .child
  • 相邻兄弟组合器（A + B）——匹配的目标元素B是 紧挨在 A的 后面 的兄弟元素。例如：p + h2
  • 通用兄弟组合器（A ~ B）——匹配的目标元素B是A 后面 的所有兄弟元素。例如：li.active ~ li

p + span仅选中<span>2</span>，但如果两者之间隔了一个元素，则该选择器不会选中任何span；p ~ span选中<span>2</span>、<span>3</span>、<span>4</span>

伪类选择器

伪类选择器用于选中处于某个特定状态的元素。这种状态可能是由于用户交互，也可能是由于元素相对于其父级或兄弟元素的位置。伪类选择器始终以一个冒号（:）开始。优先级等于一个类选择器

  • :first-child——匹配的元素是其父元素的第一个子元素
  • :last-child——匹配的元素是其父元素的最后一个子元素
  • :only-child——匹配的元素是其父元素的唯一一个子元素（没有兄弟元素）
  • :nth-child(an+b)——匹配的元素在兄弟元素中间有特定的位置。公式 an+b 里面的 a 和 b 是整数，下面给出常用例子：

  • :focus——匹配通过鼠标点击、触摸屏幕或者按 Tab 键导航而获得焦点的元素
  • :hover——匹配鼠标指针正悬停在其上方的元素
  • :root——匹配文档根元素。对 HTML 来说，这是<html>元素

更多伪类选择器可查看文档

伪元素选择器

这些选择器以双冒号（::）开头，优先级与类型选择器相等

  • ::before——例如：.menu::before
  • ::after——例如：.menu::after

必须为伪元素指定 content 属性才能让元素出现

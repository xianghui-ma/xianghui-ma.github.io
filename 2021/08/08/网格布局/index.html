网格布局

在IE10/11中使用网格布局需要加-ms-前缀。具体见 Can I Use

定义网格布局

同Flexbox，设置为 display: grid 的元素成为一个网格容器（grid container）。它的直接子元素则变成网格元素（grid items）。网格容器会表现得像一个块级元素，100%填充可用宽度

然后在网格容器中使用grid-template-columns定义网格列数和各列的宽度，使用grid-template-rows定义网格行数和各行的高度，使用grid-gap定义单元格之间的间距

上述代码定义了一个3 x 3的网格。单元格间的间距为5px。各列宽度为容器宽度减去横向总间距后剩余宽度的1 / (1fr + 1fr + 1fr) = 1 / 3。同理，各行高度为容器高度减去纵向总间距后剩余高度的1/3

<1>、本例使用了一种新单位 fr，代表每一列（或每一行）的分数单位（fraction unit）。不一定非得用分数单位，可以使用其他的单位，比如 px、em 或百分数。也可以混搭这几种单位，例如，grid-template-columns: 300px 2fr 1fr 表示第一列为固定宽度 300px，第二列分得剩余宽度的2/3、第三列分得剩余宽度的1/3 <2>、也可以grid-gap: 垂直方向间距 水平方向间距分别指定垂直与水平方向的间距

网格布局中的核心概念

前面我们已经定义了一个网格，下面我们对一个网格系统中的核心概念加以阐释

  • 网格线（即网格线条）——网格线构成了网格的框架。一条网格线可以水平或垂直，并且位于一行或一列的任意一侧。如果指定了 grid-gap 的话，它就位于网格线上
  • 网格轨道（即行/列）——一个网格轨道是两条相邻网格线之间的空间。网格有水平轨道（行）和垂直轨道（列）
  • 网格单元（即单元格）——网格上的单个空间，水平和垂直的网格轨道交叉重叠的部分
  • 网格区域（即多个单元格组成的一个矩形区域）——网格上的矩形区域，由一个到多个网格单元组成。该区域位于两条垂直网格线和两条水平网格线之间

简写方式

1、repeat()

在定义grid-template-columns与grid-template-rows时，可使用repeat(重复次数, 参数1, 参数2, ...)函数进行简写

repeat(重复次数, 参数1, 参数2, ...)表示将参数1, 参数2, ...模式重复指定的重复次数

如grid-template-rows: repeat(4, auto)等价于grid-template-rows: auto auto auto auto
grid-template-columns: repeat(3, 2fr 1fr)等价于grid-template-columns: 2fr 1fr 2fr 1fr 2fr 1fr
grid-template-columns: 1fr repeat(3, 3fr) 1fr等价于grid-template-columns: 1fr 3fr 3fr 3fr 1fr

网格布局与Flexbox的选取

网格布局与Flexbox都能很好的进行网页布局，那么我们在使用时如何选取呢？

Flexbox的布局是一维的，即通过flex-direction我们只能在横向或纵向上布局。而网格是二维的，能同时在横纵两个方向上对其元素。如下图所示：

综上，当设计要求元素在两个维度上都对齐时，使用网格。当只关心一维的元素排列时，使用Flexbox。同时网格更适合用于网页的整体布局
